requestInitialTransform(xsl, xml, parameters, success)

Have the XSLT 2.0 processor perform an "updateHTMLDocument" transformation with the provided parameters. Currently used to initiate the transform used to supply HTML for the TEI bibliography entries. Transform success handler is provided. Errors are handled by an Openbibl global error handler.
Parameters:
Name Type Description
xsl string url for xsl stylesheet
xml string url for xml source file
parameters array array of array of stylesheet parameters of form: [ [ "namespace"/null, "key", "value" ], ... ]
success function transform success callback

requestQueryTransform(xsl, xml, parameters, success)

Have the XSLT 2.0 processor perform a "transformToFragment" transformation with the provided parameters. Currently used to transform TEI XML bibliography to JSON for use in the browse/search functionality. Transform success handler is provided. Errors are handled by an Openbibl global error handler.
